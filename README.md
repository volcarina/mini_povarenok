# 🍳 Мини-поварёнок

Детское веб-приложение с простыми и весёлыми рецептами. Ребёнок выбирает блюдо, отмечает шаги готовки галочками и получает грамоту заслуженного поварёнка 🎉

> Учебный проект в рамках курса **«Разработка веб-приложений»**. В активной разработке.

---

## ✨ Что уже есть

- Каталог рецептов с эмодзи, названием и временем приготовления
- Раздел «Рекомендуем» и «Популярные»
- Боковая навигация с якорными ссылками
- Раздел «Как играть» — пошаговая инструкция для ребёнка
- Ссылки на похожие приложения
- Два CSS-варианта оформления: основной и альтернативный

## 🚀 Запуск

```bash
# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```
